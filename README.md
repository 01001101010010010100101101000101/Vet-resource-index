# Veteran Resources Directory

A comprehensive, single-page HTML directory of resources for U.S. military veterans, service members, and their families.

## What's Included

| Section | Resources |
|---|---|
| Crisis & Emergency | Veterans Crisis Line (988), Safe Helpline (MST), Domestic Violence Hotline, SAMHSA, Homeless Veterans Hotline |
| Mental Health | VA Mental Health, National PTSD Center, Telehealth apps, Give an Hour, Headstrong Project, Team RWB |
| VA Benefits & Claims | Disability compensation, VSO assistance, pension, life insurance, burial benefits, quick-reference table |
| Healthcare | VA enrollment, Community Care Network, Women's Health, dental/vision, pharmacy, caregiver program |
| Housing | VA Home Loans, HUD-VASH vouchers, SSVF rapid rehousing, SAH grants, NCHV shelter finder |
| Employment | CareerOneStop, DOL VETS/TAP, USAJOBS federal preference, VR&E, SBA Boots to Business, Hire Heroes USA |
| Education | Post-9/11 GI Bill, MGIB, Fry Scholarship, Military Tuition Assistance, Pat Tillman Foundation |
| Legal | Discharge upgrades, free legal aid, SCRA, USERRA, BVA/CAVC appeals, JAG offices |
| By State | Dropdown linking all 50 state DVA websites + state-specific benefit summaries |
| FAQ | 7 common questions on enrollment, claims, appeals, GI Bill, Guard/Reserve, survivors, DD-214 |

## Usage

No build step required. Open `index.html` directly in any modern browser, or serve it statically.

```bash
# Python quick server
python3 -m http.server 8080

# Node quick server
npx serve .
```

Then open `http://localhost:8080`.

## Features

- **Live search** — filters all resource cards instantly by keyword
- **Accessible** — skip link, ARIA labels, semantic HTML, keyboard navigable
- **Responsive** — works on mobile, tablet, and desktop
- **No dependencies** — pure HTML/CSS/JS, no frameworks or CDNs
- **Crisis-first design** — persistent 988 banner and urgent-tagged cards
